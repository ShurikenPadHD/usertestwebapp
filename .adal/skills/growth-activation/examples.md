# Growth Activation Examples

Here are realistic examples of how to use this skill:

---

## Example 1: B2B SaaS — Project Management Tool

**Product:** Kanban board for small marketing teams

**Activation Flow:**
1. Sign up (30 seconds, no credit card)
2. Create first project (prompted)
3. Add first task → \"Great, you've created your first task!\"
4. Invite teammate → \"Unlock team features\"

**Aha Moment:** User sees their first task on a board and understands the flow.

**User Loop:**
- **Trigger:** \"You have tasks due today\" (email/push)
- **Action:** Check board, move task to \"Done\"
- **Reward:** Streak counter, completion confetti
- **Investment:** More tasks = more value = harder to leave

**Retention Levers:**
- Team invites (network effect)
- Template library (setup investment)
- Weekly digest email (habit trigger)

## Example 2: Developer Tool — API Documentation Generator

**Product:** Auto-generate docs from OpenAPI specs

**Activation Flow:**
1. Connect repo via OAuth
2. Select API spec file
3. Preview generated docs
4. Deploy to custom subdomain

**Aha Moment:** User sees their API docs rendered beautifully in < 60 seconds.

**User Loop:**
- **Trigger:** Code push triggers regeneration
- **Action:** View updated docs
- **Reward:** Live preview, SEO ready
- **Investment:** Custom domain, branding

**Engagement Triggers:**
- PR comment: \"Docs updated!\"
- Weekly \"API changes this week\" email
- \"3 teams are viewing your docs\" social proof

**Early Growth Hooks:**
- \"Deploy to Vercel\" one-click button
- Shareable doc links with view counts
- Template gallery (SEO content)

## Example 3: Consumer App — Budget Tracker

**Product:** Simple budget tracker with bank sync

**Activation Flow:**
1. Sign up → \"Add your first expense\"
2. Manual entry or bank connect
3. Set a monthly budget
4. See spending breakdown

**Aha Moment:** User connects bank and sees all transactions categorized instantly.

**User Loop:**
- **Trigger:** Daily notification: \"You spent $47 today\"
- **Action:** Review transactions
- **Reward:** Budget progress ring fills up
- **Investment:** Budget goals, transaction history

**Retention Levers:**
- Savings goals (investment)
- Spending insights (value)
- Year-over-year comparison (habit)

**Metrics to Track:**
- Day 1, 7, 30 retention
- Time to first transaction
- Bank connection rate
- Monthly active users / Daily active users ratio

---

## Example 4: Developer Tool — Error Monitoring

**Product:** Lightweight error tracking for Node.js apps

**Activation Flow:**
1. npm install @errtracker/sdk
2. Copy-paste 3-line init
3. Trigger test error
4. See error in dashboard

**Aha Moment:** First error appears with full stack trace and context.

**User Loop:**
- **Trigger:** New error notification (Slack/email)
- **Action:** Investigate error
- **Reward:** One-click fix suggestion
- **Investment:** More projects = more data = baseline insights

**Growth Hooks:**
- \"Share this error\" link
- Team invites
- Public incident status page
